Name: DD OS: DOS/Win9x/Me/2k/XP
Beschreibung: DD is a replacement for the DOS "DIR" command. It will display files in one, two, three, four, or six columns, sorted, and in user-configurable colors.
Auch wenn diese Programm von 1994 ist, so kann es doch noch manchmal für denjenigen, der weiß was ein "Command prompt" nützlich sein. Gut sortiert und voller Farben werden die Dateien in kurzer DOS-Form aufgelistet.
Link zur Heimatseite: keiner
Direkt herunterladen: DD Version 4.07

Name: Microsoft TrueType core font for the Web OS: Win9x/Me/2k/XP
Beschreibung: Microsoft TrueType core fonts for the Web, dieses ist eingentlich eine Sammlung von Schriftarten, die als Standard für Netzseiten verwendet werden können. Hier ist nur die Schriftart "Andale" zur Verfügung gestellt, da sie sich als Nicht-Proprotional-Schrift als gute Alternative für "Times New Roman" und "Courier" eignet. Sie ist z.B. gut geeignet zur Darstellung von Nukleotidsequenzen, wenn man keine Serifen haben möchte.
Link zur Heimatseite: Microsoft, alternative Seite bei Sourceforge
Direkt herunterladen: Microsoft TrueType core font for the Web, Andale

Name: EditPad OS: Win9x/Me/2k/XP
Beschreibung: EditPad ist ein einfacher Texteditor mit einigen Vorteilen. Er hat nicht die Größenbeschränkung wie Notepad, kann mehrere Dateien gleichzeitig offen halten und bleibt nach dem Schließen aller Dateien in der Taskleiste aktiv und somit schnell zugänglich. EditPad ist als neuere Lite Version auch weiterhin noch frei verfügbar aber nicht mehr so klein und fein. Die Classic Version ist übrigens mit "Petite" gepackt, die neueren Versionen sind wohl mit UPX gepackt. Ein umpacken scheint möglich.
Ein Vorteil der Classic Version gegenüber der Lite Version ist, daß das automatische Erstellen von Backupdateien möglich ist. Diese Möglichkeit ist auch noch bei der Lite Version 4.5.4 vorhanden, daher habe ich diese Version hier auch noch abgelegt.
Link zur Heimatseite: EditPad Classic (deutsch)
Direkt herunterladen: EditPad Classic, englische Version 3.5.3 und deutsche Version 3.5.1 Download-Links bei Tucows.
EditPadLite v4.5.4 (englisch)

Name: Context Viewer v1.41 OS: Win9x/Me/2k/XP
Beschreibung: Context Viewer is an explorer extension in the form of a Control Panel applet that lets you preview image files and text in the right click menu. It can also display customized information about images and set hotkeys to enable or disable the preview along with a few more options.
This programme is similar to PicaView.

You can use plugins from an application named Susie. Susie by itself is a Japanese freeware image editor/viewer.
The plugins allow you then to preview pictures in the context meny (right click) of the windows explorer.

Problems:If you have installed Notepad++ the explorer context menu entry destoys the dislay as seen above. Just delete the following key in the registry to get a fast fix. 
[HKEY_CLASSES_ROOT\*\shellex\ContextMenuHandlers\ANotepad++]
Link zur Heimatseite: http://www.hoehoe.com
Official Susie Page
Susie Page with Lots of Plugins
Another download page.
Direkt herunterladen: Context Viewer, englische und japanische Version

Name: ASPI-ME OS: Win9x/Me
Beschreibung: ASPI-ME installiert die ASPI-Treiber unter Windows. Dieses ist besonders dann erwünscht, wenn man z.B. den CD-Brenner über einen ASPI-Treiber ansprechen möchte. Da Windows 9x im Original ohne ASPI-Unterstützung kommt, falls keine z.B. Adaptec- Karte eingebaut ist, ist eine Aufgradierung mit den z.B. Adaptec-Treibern auch nicht möglich. ASPI-ME installiert die Treiber und erlaubt somit danach eine Aufgradierung.
Nur für die Installation von ASPI-ME muß die Systemuhr auf 1998 zurückgesetzt werden!
Link zur Heimatseite: GRC
Direkt herunterladen: ASPI-ME
