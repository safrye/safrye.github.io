PyMOL unter Cygwin kompilieren

  • Quelldateien herunterladen von Sourceforge via Subversion 
    svn co https://pymol.svn.sourceforge.net/svnroot/pymol/trunk/pymol pymol

    Das habe ich unter Linux (OpenSuse 10.2) gemacht und die Dateien mit TotalCommander und dem Linux-drives Plugin nach Windows kopiert.
  • Cygwin aufdatieren
    und weitere Module installieren, falls noch nicht vorhanden
    • C++ (gcc or g++ package name)
    • python An interactive object-oriented scripting language
    • opengl OpenGL-related libraries
    • freeglut OpenSourced alternative to the OpenGL Utility Toolkit (GLUT) library
    • libpng12 PNG library - 1.2.x runtime
    • libpng12-devel PNG library - 1.2.x development libraries and headers
  • In Cygwin im home Verzeichnis die Datei pydistutils.cfg mit dem Inhalt 
    [build]
compiler=mingw32
    erstellen. Zum Beispiel mit dem nano Editor.
    Der Tipp kommt von hier.
  • Im pymol Verzeichnis der Quelldateien 
    python setup.py install
    starten und eine Weile warten. Es sollten jetzt keine Fehlermeldungen auftauchen. :))
  • Danach 
    python setup2.py install
    starten.
  • Kopieren der Startdatei in ein allgemeines Verzeichnis 
    cp ./pymol /usr/local/bin/pymol
    um PyMol von überall starten zu können. Diese Datei enthält 
    #!/bin/sh
/usr/bin/python /usr/lib/python2.5/site-packages/pymol/__init__.py $*
  • Pmw -- Python MegaWidgets
    installieren. Dazu die gepackte Datei in ein temporäres Verzeichnis entpacken und in Cygwin 
    python setup.py install
    in dem PMW-Quellverzeichnis (Pmw.1.3.2\src) aufrufen.
  • Nun noch einen Shortcut auf dem Desktop erstellen mit dem Ziel 
    C:\cygwin\bin\run.exe bash /usr/local/bin/pymol
    Der Weg nach Run.exe sollte natürlich entsprechend der Cygwin-Installation angepasst werden.
  • Fertig! - (Hoffentlich)
