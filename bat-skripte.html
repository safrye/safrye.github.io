Batch-Skripte

(Alle diese Skripte wurden von mir getestet und verwendet. Sie sind hier nur zur Archivierung aufgeführt. Jedwede Verwendung geschieht auf eigene Gefahr.)

Die Skripte wurden mit "Simple Source to Html Generator" in HTML überführt.
Vom Explorer aus können diese Skripte mit zusätzlichen Optionen und Übergabe von Dateinamen leicht mit Hilfe von OpenExpert gestartet werden.

Verwendungszweck

Verwendete zusätzliche Software

wgeturl.bat Datei mit wget aus URL herunterladen. wget
muscle.bat Sequenceabgleich mit muscle und Ausgabe in Genedoc. muscle, GeneDoc, msgbox
dir2arc_lmod.bat Mehrere Unterverzeichnisse in einzelne Archiv umpacken. Start z.B. mit "drag and drop" des darüber liegenden Verzeichnisses. msgbox, arj
mp3-verz-eindampfen.bat Dateigröße von MP3-Dateien verringern, z.B. mehr Dateien auf einem MP3-Spieler mit geringer Kapazität unter zu bringen. (Version 0.1) lame, msgbox, wprompt
rexopenw.bat OpenExpert Einstellungen speichern. OpenExpert
ORDIKONE.bat Ikone einer Datei dem zugehöringen Ordner zuordnen. 4DOS
rfinder.bat Start the Perl script repeat_finder.pl for a range of repeat sizes. msgbox, wprompt, winput
sekalign.bat Use sequential profile alignment to align sequences one by one to a premade alignment. Useful to align several shorter sequences that do not cover the same area of a bigger sequence. E.g.: Align results from sequencing reactions with differnt primers to one big reference sequence. ClustalW
