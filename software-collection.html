Science related Programs

Biology

Name: ClustalW OS: DOS
Properties: Sequence Alignment Program which does a pretty good job. Runs very well in the DOS-Box under Windows 95 and gives you back the feeling of the old DOS days. It's easy to handle and free.
Link to Homepage: http://www.ebi.ac.uk/ebi_home.html
Link to Downloadpage: ftp://ftp.ebi.ac.uk/pub/software/dos/

Name: Macaw OS: Win3.x Win32s
Properties: Sequence alignment program with graphical interface. Very good to get a quick overview of similarities in different sequences. Unfortunately the program is not running very stable so that it is a good to save the project after each important step.
Link to Homepage: ftp://ncbi.nlm.nih.gov/pub/macaw

Name: TreeCon OS: Win95
Properties: Phylogenetic Tree drawing program which allows you to import your alignments from other programs. The program allows you to edit the Tree in a very user friendly way.
Link to Homepage: Homepage Yves Van de Peer

Name: TreeView OS: Win95 Win3.x Mac
Properties:
Link to Homepage: TreeView Homepage

Name: Anthepro OS: Win
Properties: Analysis program for protein sequences. Does also antigenecity predicton. Comes with a 3D viewer for protein structures. Includes ClustalW, FastA and RasWin (not allways the news versions).
Link to Homepage: http://www.ibcp.fr/ANTHEPROT/Documentation_antheprot.html

Name: The Protein Analyis Work Sheet (PAWS) OS: Win95
Properties: Does what the name allready says. Gives you useful information about your protein sequence. Can also read the sequence.
Link to Homepage: The Protein Analyis Work Sheet (PAWS) homepage

Name: Protein Sequence Analysis OS: Win95
Properties: Does what the name allready says but can additonally be used for DNA sequence editing. The only as (crippled) shareware program available (as I know) which converts DNA to Protein.
Link to Homepage:
Ira W. Palmer
36 Mt. Laurel Ct.
Gaithersburg, MD. 20879

Chemistry

Name: SoftShell or ChemWeb OS: Win3/9x/2k/XP
Properties: This is a free drawing program for chemical structures. It is/was hosted by BioRad. It's quite old, from 1995, but is still useful for drawing some basic chemical structures.
Link to Homepage: SoftShell Online
By now (2002) this program seams to be not longer available on the net. Therefor I put a copy of the installation program for version 3.1.4 here.

Graphic Programs

Graphic Viewers

Name: Slowview / Bennig's OS: Win9xNT
Properties: Excellent viewer, Bennigs's is also useful for other multimedia files and is extendable with plugins. Install Ogg DSF to use it also for ogg mucis files.
Good replacement for e.g. ACDSee Classic with a simelar GUI.
Link to Homepage: www.slowview.at
Slowview changed to Bennig's - Bennig's changed to Ashampoo Photo Commander (commercial)
You may find Slowview through Googles and Bennig's at BetaNews (german version, english version).

Name: IrfanView OS: Win9xNT
Properties: The best FREEWARE viewer and a very good alternity to ACDSee. Don't forget to install the plugins which also come as a single big package.
Link to Homepage: IrfanView Homepage


Name: XNView OS: Win Unix
Properties: FREEWARE viewer with support for MANY graphic file formats.
Link to Homepage: XNView Homepage


Name: ACDSee OS: Win95 WinNT Win3.x
Properties: A very fast graphic viewer with Filetree, Slideshow and converter for some graphic formates. A MUST HAVE!
Link to Homepage: ACDSee Homepage


Name: PicaView OS: Win95 WinNT Win3.x
Properties: It's the little brother of ACDSee and gives you a picture previewer on the right mouseclick in the Windows Explorer (thumbnails in the context menu) and most of the functions of ACDSee but no slideshow.
Link to Homepage: ACDSee Homepage

Name: WinJPEG OS: Win3.x
Properties: Only a 16 bit application but it does a very good job in dithering, has a batch conversion option, does slideshows and runs without any known problems under Win95. It has also many picture manipulation functions so it's also a good picture editor. Actually, this was the first Windows based graphic program I used and I still use it!
Link to Homepage: PixVision Homepage
WinJPEG
PMJPEG ("32 bit version")

Name: Thumbs OS: Win95
Properties: This program is great when you have a big collection of pictures distributed all over your harddrive. It shows the filestructure in a tree and indicates where the pictures are located. Little preview pictures (thumbs) are used to show the pictures. The program also allows a the editing of pictures and can gererate contact sheets for your webpage. The only disadvantage I experienced with this program are the long loading times with big picuture collections obviously due to the fact that the thumbs are stored in one big datafile in the program folder.
Link to Homepage: Cerious Software Inc.
