Name: StarCalc OS:Win9x/Me/2k/XP
Properties:
StarCalc is the fastest professional astronomy planetarium & star mapping program for Windows 9x/ME/NT/2000/XP (PocketPC version available). It illustrates star positions of any instance of the day observed from any geographic locations on the Earth. The star positions can be viewed and presented as images of hemispherical whole sky or any of the user defined sub-areas. These images can be zoomed at different scales, rotated, screen-captured and printed. Also, StarCalc has many other functions of calculation and showing of sky objects. Best of all StarCalc is freeware.
Link to Homepage: StarCalc

Name: Cartes du Ciel / Sky Charts OS:Win9x/Me/2k/XP
Properties: This program enables you to draw sky charts, making use of the data in 16 catalogs of stars and nebulae. In addition the position of planets, asteroids and comets are shown.
The purpose of this program is to prepare different sky maps for a particular observation. A large number of parameters help you to choose specifically or automatically which catalogs to use, the colour and the dimension of stars and nebulae, the representation of planets, the display of labels and coordinate grids, the superposition of pictures, the condition of visibility and more. All these features make this celestial atlas more complete than a conventional planetarium.
Link to Homepage: Cartes du Ciel

Name: Horodruin OS:Win9x/Me/2k/XP
Properties:
This program allows you to synchronize more than 2 folders with just few clicks. It's very simple, but you first must read the included documentation. It has the following key features:
  • multiple folder handling (2 or more).
  • the folder analysis can be modified on your needs.
  • multithreaded.
  • optimal file multithreaded copy engine (by using temporary file and/or by overwriting files).
  • configurable. You can easily integrate it with a text editor, a hex editor, and a diff viewer.
  • minimal registry usage (only for Explorer integration).
  • small, fast, and clean.
  • HTML Help included.
  • for advanced users only!!!

Name: WSize OS:Win9x/Me/2k/XP
Properties: From the author of Saver Starter there is a program called Sizer (new homepage) for resizing any window to an exact, predefined size. A none-GUI command line version is included with his Hot Key Plus, another hot key configuration program, and called wsize.exe. WSize can easily be used with HoeKey by putting for example 
=Rem--- Fenstergröße *** Win+ --------
~6=Run|c:\bin\wsize.exe 640 480     ; Fenster 640 x 480
~7=Run|c:\bin\wsize.exe 800 600     ; Fenster 800 x 600
~8=Run|c:\bin\wsize.exe 1024 768    ; Fenster 1024 x 768
~9=Run|c:\bin\wsize.exe 1200 950    ; Fenster 1200 x 950
~0=Run|c:\bin\wsize.exe 1280 1024   ; Fenster 1280 x 1024
into your hoekey.ini file.
Link to Homepage: Hot Key Plus
