Uninstallation problem

On some systems the uninstaller does not remove all registry keys so that the program is gone by the right-click Explorer extension menue is still visible.
To remove this search in the registry for all occurences of the key {A4D78B20-6E05-1069-8758-4E73FD83DEAD} and remove them. This tip was taken from here.
Link to homepage: http://www.ranvik.net/totalcopy (not active anymore)
Direct download: Look for the Zip icons above!
