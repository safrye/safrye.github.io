Things are stored here just for collective purposes.

  • WinPos, position and resize a window on the desktop
  • wsize, just resize the active window on the desktop. This program originates from the juggler saver webpage.
    use like this: 
    c:\>wsize.exe 800 600
  • ls, by Alan Klietz
  • ls32, from www.klabaster.com
  • dedit, Description Editor, by Mark Incley
  • CmdTime3, synchronyze the time with an Internet Time Server, by Jury Gerasimov
  • LE, list executables
  • sleep, wait for a specified time
  • tppatch, patch old Turbo Pascal file to avoid the clocking error
  • which, GNU utility to show which executable will be started from the path
  • Pushkeys, Sends one or more keystrokes to the active window, even DOS applications, as if typed at the keyboard.
  • less, more than more, by Mark Nudelman,
    lesskey - specify key bindings for less
  • LIST, by Vernon D. Buerg
  • MCD, make and change to directory
  • PsUptime v1.1, from Mark Russinovich
